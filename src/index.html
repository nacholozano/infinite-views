Web clone of Google Material Tabs trying to imitate native apps' behavior.
Made in VanillaJS with requestAnimationFrame and CSS 2D transforms.

Built with ionic2 in mind so Better test touch events in mobile.
At the moment is just plain js, not typescript or angular.
Good performance in Android 5.0-Chrome 57. Same technology that android WebView.

Features:
  • Change tab with touch event.
  • Change tab in any moment.
  • It works with mobile orientation change.
  • The current and next/previous tab is always visible.
  • Load tab data when user changes tab.
    (No request cancellation and not refresh at the moment)
  • Two modes:
    • Many tabs with auto width and scrollable element (This demo).
      Video example
    • Few tabs with same with.
      Photo example
      See usage in github.
